# Папка Branch (Для устройства на кассе/в филиале)

Этот скрипт должен постоянно работать на компьютере, расположенном непосредственно в филиале. Он получает команды от центрального бота через HTTP и эмулирует нажатия клавиш для выбранного кода.

## 🚀 Инструкция по установке

### 1. Подготовка
Скачайте эту папку `branch` на компьютер филиала. Убедитесь, что установлен Python 3.

### 2. Настройка файла `.env`
В папке `branch` создайте файл `.env`. Если его нет, переименуйте файл `.env.example` в `.env`.

Откройте его блокнотом и пропишите:
```env
# Идентификатор филиала, ровно такой, как записан в admin/config.py
BRANCH_ID=branch_1

# Внешний айпи адрес сервера с запущенным ботом
BOT_SERVER_URL=http://YOUR_SERVER_IP:8000
```
*(Не забудьте поменять `YOUR_SERVER_IP` на реальный IP адрес сервера Ubuntu)*

### 3. Установка библиотек
Откройте консоль/PowerShell из этой папки и выполните:
```powershell
pip install -r requirements.txt
```

### 4. Запуск агента

Есть два способа запуска файла:

**Способ А (С консолью, для проверки):**
Запустите в консоли чтобы увидеть, как всё работает:
```powershell
python branch_agent.py
```

**Способ Б (Скрытый фоновый режим, как AHK):**
Чтобы агент работал постоянно и не мозолил глаза консолью (окно не появится):
Просто **дважды кликните** по файлу `start_hidden.vbs` (или запустите его).
Скрипт будет работать в фоне 24/7. Если вам потребуется его выключить — откройте **Диспетчер задач** (Ctrl+Shift+Esc), найдите процесс `pythonw.exe` и завершите его.
